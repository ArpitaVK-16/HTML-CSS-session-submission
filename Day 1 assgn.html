All about HTML...

HTML stands for Hypertext Markup Language. It's not the programming language primarily. It's preferably used fr building website. Since it's not a programming language we cannot perform any calculation using HTML



Short history of HTML

HTML is the language of the Internet. HTML was orginally designed for sharing scientific documenmts. Adoptation over the years made it suitable to describe the other types of documents that can be displyed as web pages on the Internet.


HTML elements:

  • Building blocks of an HTML pages

  • Pieces of content such as paragraph, list and tables

  • Represented by tags

  • Browsers use tags to render the content

There are different types of HTML5 elemets:
Structural elements, Graphics and embedded content, input elements, web storage, web workers etc.
